### WARNING

This tool permanently destroys files. Once a file is dragged into Nuker3000 and confirmed, it will be securely overwritten and deleted. There is no recovery, no undo, no recycle bin. Use at your own risk.

### What Is Nuker3000?

Nuker3000 is a simple drag-and-drop secure file shredder for Windows. Designed to look clean and modern while staying extremely easy to use. Drag a file into the window, confirm, and watch the deletion sequence.

### Features

* Drag & drop deletion
* Multi-language UI (English / Czech)
* Secure overwrite (3 passes)
* Animated deletion sequence
* Clean one-window interface
* No console window
* Compact single-file EXE
* Subtle signature: Made by William Klier — 2025

### How It Works

1. Overwrites the entire file with random data
2. Deletes it from disk
3. Displays the deletion animation
4. Removes any remaining traces

### Installation

* Download the Nuker3000.exe or installer
* Run the application
* Drag a file into the window
* Confirm the action

The author is not responsible for any lost data. Deleting files with Nuker3000 is permanent.
